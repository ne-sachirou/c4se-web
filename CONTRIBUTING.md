CONTRIBUTING
==
Requirments
--
- PHP >=5.4
- Composer
- Latest Node.js & NPM
- Gulp
- Bower
- Babel

```sh
composer install
npm install
bower install
```

Test
--
`npm test`

Dev Server
--
`npm start`

Build
--
`bin/gulp build`

Automatic building & testing while editing.

`bin/gulp watch`

Deploy
--
You must have an authorized SSH key.

`bin/gulp deploy`
