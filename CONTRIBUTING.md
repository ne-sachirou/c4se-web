CONTRIBUTING
==
Requirments
--
- PHP >=5.6
- Composer
- Latest Node.js & NPM
- Gulp
- Bower
- Linux-like system & bash
- FontForge

```sh
composer install
npm install
bower install
```

Test
--
`npm test`

Dev Server
--
`npm start`

Build
--
`gulp build`

Automatic building & testing while editing.

`gulp watch`

Deploy
--
You must have an authorized SSH key.

`gulp deploy`
